üìñ Configuration Guide

SoftAim Configuration

// Basic SoftAim setup cl_softaim 1 // Enable SoftAim cl_softaim_strength 1.0 // Assistance strength (0.5-2.0) cl_softaim_maxdist 512 // Maximum targeting distance cl_softaim_fov 90 // Field of view for targeting cl_softaim_smooth 0.3 // Smoothing factor cl_softaim_autoswitch 0 // Auto weapon switching (experimental)

cl_rDrive Configuration

// Randomized SOCD setup cl_rDrive 1 // Enable randomized SOCD cl_rDrive_ms 5 // Max delay in milliseconds cl_rDrive_pairs "forward,back;left,right" // Configure key pairs // Advanced configuration cl_rDrive_pairs "forward,back;left,right;attack,attack2;lookup,lookdown"
‚ö†Ô∏è Important: Only enable one SOCD system at a time. Use either cl_iDrive OR cl_rDrive, not both simultaneously.

üîÑ How cl_rDrive Works

  1. Conflict Detection: When both keys in a configured pair are pressed simultaneously
  2. Random Selection: The system randomly chooses which key to clear (50/50 chance)
  3. Random Timing: A random delay (0 to cl_rDrive_ms milliseconds) is applied
  4. Input Clearing: The selected key's input is cleared after the delay
  5. State Reset: When both keys are released, the system resets
