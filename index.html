🎯 Softaim Configuration

This build includes advanced softaim features. Configure these console variables to customize the behavior to your preferences. Open the console (~ key) and enter these commands:

cl_softaim "1"
Enables or disables the softaim feature. Set to "1" to enable, "0" to disable.
cl_softaim_fov "30"
Field of view for softaim activation in degrees. Lower values require more precise aim, higher values are more forgiving.
cl_softaim_lg_range "600"
Maximum effective range for Lightning Gun softaim. Targets beyond this distance won't be affected.
cl_softaim_autoswitch "1"
Automatically switches from Lightning Gun to rocket launcher when enemy is out of LG range. Very useful for maintaining pressure.
cl_softaim_maxdist "600"
Maximum distance for softaim to engage targets. Helps prevent unwanted aim assistance on distant enemies.
cl_softaim_smoothness "0.6"
Controls how smoothly the aim adjusts (0.0 to 1.0). Lower values = more responsive, higher values = smoother transitions.
cl_softaim_strength "40"
Overall strength of the softaim effect. Experiment with this value to find what works best for you! Start with 40 and adjust up or down.
💡 Pro Tip: Create a config file (e.g., softaim.cfg) with these settings and execute it with exec softaim.cfg in the console. This way you can quickly switch between different configurations or restore your preferred settings.
